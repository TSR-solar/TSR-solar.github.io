使用说明

点击Escape from MAZE文件夹里面的Escape from MAZE应用程序即可运行；

若没有找到Escape from MAZE.exe，可以使用Visual Studio打开Escape from MAZE.sln，按CTRL+F5运行；

查看源代码可以使用Visual Studio打开Escape from MAZE.sln，查看源文件main.cpp；

常见问题及解决方法

若无法运行应用程序，请先关闭上一次运行打开的终端；再重新尝试；
若仍无法运行，且显示项目里面有错误，尝试点击Visual Studio项目上面的项目，选择“Escape From Maze和属性”，在“C/C++”目录里面选择“预处理器”，在预处理器里面点击“编辑”，在最后一行加上_CRT_SECURE_NO_WARNING；然后点击确定，再重新运行；

项目内容

主要功能

  1. 运行Escape from MAZE.exe 后，显示开始页面，内有4个按钮，通过捕获鼠标信息来检测按钮是否被按下，从而执行接下来的操作。
  2. 游戏设置里可以调整迷宫大小；游戏说明里面有帮助。
  3. 点开始后，会绘制游戏图片到另一个image中；该过程在.exe运行直到结束不会进行第二次。
  4. 弹出对话框提醒用户输入长宽（不能超出约定范围）；若此前已经设置过则不会弹出此提示。
  5. 使用动态内存分配分配迷宫内存；当游戏结束后迷宫内存会被释放。该过程涉及了C++相关内容。
  6. 使用广度优先搜索生成迷宫元素，包括墙、空地、起点、终点；用函数rand()、srand()和time随机生成迷宫，迷宫保证有唯一解。
  7. 根据迷宫生成的数据在对应的地方绘制元素对应的游戏图片，游戏过程中由于涉及视角转换所以每0.1s都会重新绘制。
  8. 记录起始时间，此后每0.1s记录一次时间，将其与开始时间之差显示在屏幕右上角。
  9. 捕获键盘数据来实现玩家的移动和标记、取消标记操作，以及返回开始界面，每0.1s才会捕获一次，以免玩家移动过快；
  10. 玩家移动时，不会穿墙。
  11. 到达终点后，会询问用户是否重新开始；如果不重新开始则返回主界面；
  12. 按ESC和在初始界面点击退出游戏都会询问，以免误触；

设计原理

  1. 文件的拓展名是.cpp而不是.c，是因为EasyX的库、动态内存分配、迷宫生成中涉及到了C++的内容；
  2. 由于Visual Studio的安全问题，有关TCHAR的字符串操作必须要在预处理器里面加入宏_CRT_SECURE_NO_WARNING；
  3. 在等待用户操作、主界面的离开与返回、游戏进行阶段使用循环结构，只有达到了特定的条件才会用break结束循环；
  4. 将迷宫元素和键盘操作数据常量均封装在枚举数据结构中，将所有按钮封装在结构体数据结构中，提高程序易读性；
  5. 将迷宫图片提前绘制在g_imgItem中缓存，不会显示在游戏界面；
  6. 迷宫生成方面，先把整个地图变成墙，再采用广度优先搜索算法生成路（把墙变成空地）；
    每次搜索前随机打乱遍历方向，如果路径之间可以联通就停止搜索；因此保证了随机的迷宫有唯一的解；
  7. 视野会随着玩家的移动而转换，防止玩家走到屏幕外面，但不会移出地图。因此地图必须不停地绘制；
  8. 捕获鼠标信息方面采用了easyx.h里的getmessage()函数，主要用于检测按钮是否被按下；当鼠标移到按钮上时按钮会高亮显示；如果同时按下左键就会执行按钮相应功能；
  9. 捕获键盘信息方面采用了WinUser.h里的GetAsyncKeyState()函数；并采用位运算处理按下多个按键的情形，最后使用DispatchCmd()函数处理这些信息并执行相关功能；
  10. 采用sleep(100)来保证每0.1s才捕获一次键盘信息；防止玩家移动速度过快；
  11. 游戏开始时记录起始时间，此后每0.1s记录一次时间，将其与开始时间之差显示在屏幕右上角（计时器）。
  12. 使用WinUser.h里的MessageBox来生成对话框，包含输入数据、选择是/否等功能；
  13. 开始界面通过for循环连续绘制矩形的功能，绘制了一个渐变色的外框；

项目总结

这是我首次利用程序设计制作游戏，其实很多东西很不熟悉，包括不知道图形库里的函数怎么用，迷宫是怎样生成的，如何读取鼠标和键盘操作，如何控制视角转换，如何进入和退出游戏。

总之，从0开始确实困难，因此直接看了网上现有的代码，然后自己绘制了一个开始页面，并调大了游戏的窗口，然后把墙体、角色以及标记的图案换了一下，其它像绘制迷宫、视角移动、迷宫生成、内存管理、读取鼠标和键盘信息都是现有的。。总之我并没有做什么工作。。。

但是也对源代码进行了深入分析，现已读懂所有内容，知道每一步要干什么了。

虽然现在这个项目还有许多未实现的功能，也有一些地方不尽人意，比如无法缩放窗口，迷宫难度太大，没有倒计时功能等等，但是总体来说，我基本掌握了EasyX图形库的基本功能，也让我的C语言的基本语法、数据结构、算法有了实战的机会。

所以还会有下次的，把这次的不足弥补回来。

#pragma once
#include <graphics.h>				// EasyX 绘图工具头文件
#include <time.h>					// 用于计时器和生成随机数头文件


//全局变量与宏
#define LENTH 1600					// 窗口的长度
#define WIDTH 960					// 窗口的宽度
#define ROUNDRATE 30				// 按钮圆弧的弧度
#define STRLEN 50					// 按钮文字的字符串长度

ExMessage m;						// 记录鼠标信息的结构体
BYTE** g_aryMap = NULL;				// 迷宫地图
IMAGE	g_imgItem(360, 40);			// 地图元素
IMAGE	g_imgSight(1600, 960);		// 游戏的视野（与窗口大小一致）
POINT	g_ptPlayer;					// 游戏者的位置
RECT	g_rtSight;					// 游戏的视野的范围
SIZE	g_szMap;					// 迷宫地图的尺寸

// 枚举地图元素，兼做元素位置的 x 坐标
enum MAPITEM { MAP_WALL = 0, MAP_PLAYER = 20, MAP_GROUND = 40, MAP_MARK = 80, MAP_ENTRANCE = 120, MAP_EXIT = 140, MAP_OUTSIDE = 160 };
enum CMD { CMD_QUIT = 1, CMD_UP = 2, CMD_DOWN = 4, CMD_LEFT = 8, CMD_RIGHT = 16, CMD_MARK = 32, CMD_CLEARMARK = 64 };

typedef struct tagBUTTON	// 定义按钮结构体
{
	int left;					// 按钮的左部
	int top;					// 按钮的顶部
	int right;					// 按钮的右部
	int bottom;					// 按钮的底部
	TCHAR str[STRLEN];			// 按钮的文字提示
} button;
button start, set, help, difficulty, back, quit;				// 定义游戏中的所有按钮

clock_t tStart, tNow, tPass;	// 记录起始、现在和经过的时间					
int Game;						// 记录游戏状态（欢迎、开始、退出）							


// 函数声明
int welcome();							// 显示初始界面
int checkButton(button btn);			// 检测按钮是否被按下
void Setting();							// 显示设置界面
void Help();							// 显示帮助界面
void initImage();						// 初始化游戏图片
void initGame();						// 初始化游戏数据
void GetMazeSize();						// 提示用户输入迷宫大小
void MakeMaze(int width, int height);	// 生成迷宫：初始化（注意：宽高必须是奇数）
void TravelMaze(int x, int y);			// 生成迷宫：遍历(x,y)四周
MAPITEM GetMazeItem(int x, int y);		// 获取指定坐标的迷宫元素
void Paint();							// 绘制视野范围内的迷宫
int GetCmd();							// 获取用户输入的指令
void DispatchCmd(int cmd);				// 处理用户输入的指令
void OnUp();							// 向上移动
void OnDown();							// 向下移动
void OnLeft();							// 向左移动
void OnRight();							// 向右移动
void OnMark(MAPITEM value);				// 在地图中做标记
bool CheckWin();						// 检测是否到出口以及是否再来一局
bool Quit();							// 询问用户是否退出游戏
bool EndGame();							// 询问用户是否结束游戏并返回主界面


//函数定义


//主函数
int main()
{
	initgraph(LENTH, WIDTH);			// 创建绘图窗口
	srand((unsigned)time(0));			// 用时间设置随机数种子
	int error = 0;


	// 窗口页面循环，若离开循环则关闭窗口
	while (1)	
	{
		// 初始界面循环，若离开循环则退出初始界面
		while (1)
		{
			cleardevice();					// 清屏

			// 通过初始页面按钮给 Game 赋值，决定要跳转的窗口
			Game = welcome();				
			if (Game == 1 || Game == -1)
			{
				break; //如果 Game 为 1 则开始游戏，为 -1 则退出游戏 
			}
		}

		// 退出游戏
		if (Game == -1)
		{
			break;
		}

		cleardevice();	// 清屏
		if (error == 0)
		{
			initImage();	// 初始化游戏图片
			error = 1;
		}
		initGame();		// 初始化游戏数据

		// 游戏过程
		int c;				
		tStart = clock();	// 记录游戏开始的时间
		while (!(((c = GetCmd()) & CMD_QUIT) && EndGame()))	//如果按 ESC 则询问是否返回初始界面
		{
			tNow = clock();	// 记录当前的时间

			// 根据开始时间和当前时间判断游戏进行的时间，并转换为以秒为单位
			tPass = (int)((tNow - tStart) / CLOCKS_PER_SEC);	
			DispatchCmd(c);	// 检测按下的按键
			Paint();		// 实时 绘制游戏画面
			if (CheckWin()) // 如果到出口就询问是否再来一局，否则返回初始界面
			{
				break;
			}

			// 延时0.1s
			Sleep(100);
		}
		cleardevice();		// 清屏
	}

	closegraph();			// 关闭窗口
	return 0;
}

// 显示初始界面
int welcome()
{
	// 打印游戏名称及作者
	settextstyle(114, 0, _T("Consolas"));
	settextcolor(RED);
	outtextxy(350, 200, _T("Escape from MAZE"));

	settextstyle(50, 0, _T("Consolas"));
	settextcolor(RED);
	outtextxy(1100, 350, _T("By solxr"));

	// 绘制渐变色外框（背景）
	for (int i = 0; i < 256; i++)
	{

int checkButton(button btn)
{

	RECT r = { btn.left,btn.top + 15,btn.right,btn.bottom };

	// 如果鼠标在按钮里面，则高亮显示按钮
	if (m.x >= btn.left && m.x <= btn.right && m.y >= btn.top && m.y <= btn.bottom)
	{
		setlinecolor(LIGHTRED);
		settextcolor(LIGHTRED);
		roundrect(btn.left, btn.top, btn.right, btn.bottom, ROUNDRATE, ROUNDRATE);
		drawtext(btn.str, &r, DT_CENTER);

		// 如果按下左键，则执行按钮内容
		switch (m.message)
		{
			case WM_LBUTTONDOWN:
			{
				return 1;
			}
		}
	}
	// 如果鼠标不在按钮里面，则恢复原来的颜色
	else
	{
		setlinecolor(RED);
		settextcolor(RED);
		roundrect(btn.left, btn.top, btn.right, btn.bottom, ROUNDRATE, ROUNDRATE);
		drawtext(btn.str, &r, DT_CENTER);
	}
	return 0;
}

// 初始化游戏数据
void initGame()
{
	// 提示用户输入迷宫大小(如果没有调整难度的话)
	if (g_szMap.cx == 0 || g_szMap.cy == 0)
	{
		GetMazeSize();
	}

	// 初始化参数
	if (g_aryMap != NULL)
	{	// 清理迷宫地图占用的内存
		for (int x = 0; x < g_szMap.cx + 2; x++)
			delete[] g_aryMap[x];
		delete[] g_aryMap;
	}
	MakeMaze(g_szMap.cx, g_szMap.cy);		// 创建迷宫
	g_ptPlayer.x = 2;						// 设置游戏者的位置
	g_ptPlayer.y = 2;
	g_rtSight.left = 0;						// 设置视野范围
	g_rtSight.top = 0;
	g_rtSight.right = 80;
	g_rtSight.bottom = 48;

	// 绘制游戏区
	Paint();
}


// 提示用户输入迷宫大小
void GetMazeSize()
{
	g_szMap.cx = g_szMap.cy = 0;

	// 获取用户输入的宽高
	TCHAR s[4];
	while (g_szMap.cx < 80 || g_szMap.cx > 150)
	{
		InputBox(s, 4, _T("请输入Maze的长度\n范围：80～150(默认为100)"), _T("输入"), _T("100"));
		g_szMap.cx = _ttoi(s);
	}
	while (g_szMap.cy < 40 || g_szMap.cx > 150)
	{
		InputBox(s, 4, _T("请输入Maze的宽度\n范围：48～150(默认为60)"), _T("输入"), _T("60"));
		g_szMap.cy = _ttoi(s);
	}

	// 确保宽高为奇数
	if (g_szMap.cx % 2 != 1) g_szMap.cx++;
	if (g_szMap.cy % 2 != 1) g_szMap.cy++;
}


// 生成迷宫：初始化（注：宽高必须是奇数）
void MakeMaze(int width, int height)
{
	// 确保宽高为奇数
	if (width % 2 != 1 || height % 2 != 1)
		return;

	int x, y;

	// 定义迷宫尺寸，并分配迷宫内存
	g_aryMap = new BYTE * [width + 2];
	for (x = 0; x < width + 2; x++)
	{
		g_aryMap[x] = new BYTE[height + 2];
		memset(g_aryMap[x], MAP_WALL, height + 2);
	}

	// 定义边界
	for (x = 0; x <= width + 1; x++)
		g_aryMap[x][0] = g_aryMap[x][height + 1] = MAP_GROUND;

	for (y = 1; y <= height; y++)
		g_aryMap[0][y] = g_aryMap[width + 1][y] = MAP_GROUND;

	// 定义入口和出口( 左上和右下 )
	g_aryMap[1][2] = MAP_ENTRANCE;
	g_aryMap[width][height - 1] = MAP_EXIT;

	// 从任意点开始遍历生成迷宫
	TravelMaze(((rand() % (width - 1)) & 0xfffe) + 2, ((rand() % (height - 1)) & 0xfffe) + 2);

	// 将边界标记为迷宫外
	for (x = 0; x <= width + 1; x++)
		g_aryMap[x][0] = g_aryMap[x][height + 1] = MAP_OUTSIDE;

	for (y = 1; y <= height; y++)
		g_aryMap[0][y] = g_aryMap[width + 1][y] = MAP_OUTSIDE;
}


// 生成迷宫：遍历 (x, y) 四周
void TravelMaze(int x, int y)
{
	// 定义遍历方向
	int d[4][2] = { 0, 1, 1, 0, 0, -1, -1, 0 };

	// 将遍历方向乱序
	int n, t, i;
	for (i = 0; i < 4; i++)
	{
		n = rand() % 4;
		t = d[i][0], d[i][0] = d[n][0], d[n][0] = t;
		t = d[i][1], d[i][1] = d[n][1], d[n][1] = t;
	}

	// 尝试周围四个方向
	g_aryMap[x][y] = MAP_GROUND;
	for (i = 0; i < 4; i++)
		if (g_aryMap[x + 2 * d[i][0]][y + 2 * d[i][1]] == MAP_WALL)
		{
			g_aryMap[x + d[i][0]][y + d[i][1]] = MAP_GROUND;
			TravelMaze(x + d[i][0] * 2, y + d[i][1] * 2);		// 递归
		}
}


// 获取指定坐标的迷宫元素
MAPITEM GetMazeItem(int x, int y)
{
	return (MAPITEM)g_aryMap[x][y];
}


// 绘制视野范围内的迷宫
void Paint()
{
	int x1, y1;
	SetWorkingImage(&g_imgSight);

	// 绘制视野内的迷宫
	for (int x = g_rtSight.left; x <= g_rtSight.right; x++)
	{
		for (int y = g_rtSight.top; y <= g_rtSight.bottom; y++)
		{
			x1 = (x - g_rtSight.left) * 20;
			y1 = (y - g_rtSight.top) * 20;
			putimage(x1, y1, 20, 20, &g_imgItem, GetMazeItem(x, y), 0);
		}
	}

	// 绘制游戏者
	x1 = (g_ptPlayer.x - g_rtSight.left) * 20;
	y1 = (g_ptPlayer.y - g_rtSight.top) * 20;
	putimage(x1, y1, 20, 20, &g_imgItem, MAP_PLAYER, 0);

	// 绘制计时器
	settextstyle(40, 0, _T("黑体"));
	settextcolor(RED);
	TCHAR strTime[STRLEN];
	_stprintf(strTime, _T("%lds"), tPass);
	outtextxy(1500, 50, strTime);

	// 更新到绘图窗口
	SetWorkingImage(NULL);
	putimage(0, 0, 1600, 960, &g_imgSight, 10, 10);
}

// 获取用户输入的命令
int GetCmd()
{
	int c = 0;

	// 使用位运算节省空间，同时处理多个按键；
	if (GetAsyncKeyState(VK_LEFT) & 0x8000)		c |= CMD_LEFT;
	if (GetAsyncKeyState(VK_RIGHT) & 0x8000)		c |= CMD_RIGHT;
	if (GetAsyncKeyState(VK_UP) & 0x8000)		c |= CMD_UP;
	if (GetAsyncKeyState(VK_DOWN) & 0x8000)		c |= CMD_DOWN;
	if (GetAsyncKeyState('A') & 0x8000)		c |= CMD_LEFT;
	if (GetAsyncKeyState('D') & 0x8000)		c |= CMD_RIGHT;
	if (GetAsyncKeyState('W') & 0x8000)		c |= CMD_UP;
	if (GetAsyncKeyState('S') & 0x8000)		c |= CMD_DOWN;
	if (GetAsyncKeyState(' ') & 0x8000)		c |= CMD_MARK;
	if (GetAsyncKeyState('C') & 0x8000)		c |= CMD_CLEARMARK;
	if (GetAsyncKeyState(VK_ESCAPE) & 0x8000)		c |= CMD_QUIT;

	return c;
}


// 处理用户输入的命令
void DispatchCmd(int cmd)
{
	// 将按键转换为操作再转换为函数
	if (cmd & CMD_UP)			OnUp();
	if (cmd & CMD_DOWN)			OnDown();
	if (cmd & CMD_LEFT)			OnLeft();
	if (cmd & CMD_RIGHT)		OnRight();
	if (cmd & CMD_MARK)			OnMark(MAP_MARK);
	if (cmd & CMD_CLEARMARK)	OnMark(MAP_GROUND);
}


// 向上移动
void OnUp()
{
	// 上方有空间且不是墙壁
	if (g_ptPlayer.y > 1 && GetMazeItem(g_ptPlayer.x, g_ptPlayer.y - 1) != MAP_WALL)
	{
		g_ptPlayer.y--;

		// 当移动到边缘时，挪动视野，以防游戏者离开视野
		if (g_ptPlayer.y - g_rtSight.top < 8 && g_rtSight.top > 0)
		{
			g_rtSight.top--;
			g_rtSight.bottom--;
		}
	}
}


// 向左移动
void OnLeft()
{
	if (g_ptPlayer.x > 1 && GetMazeItem(g_ptPlayer.x - 1, g_ptPlayer.y) != MAP_WALL && GetMazeItem(g_ptPlayer.x - 1, g_ptPlayer.y) != MAP_ENTRANCE)
	{
		g_ptPlayer.x--;

		if (g_ptPlayer.x - g_rtSight.left < 10 && g_rtSight.left > 0)
		{
			g_rtSight.left--;
			g_rtSight.right--;
		}
	}
}


// 向右移动
void OnRight()
{
	if (g_ptPlayer.x < g_szMap.cx && GetMazeItem(g_ptPlayer.x + 1, g_ptPlayer.y) != MAP_WALL)
	{
		g_ptPlayer.x++;

		if (g_rtSight.right - g_ptPlayer.x < 10 && g_rtSight.right <= g_szMap.cx)
		{
			g_rtSight.left++;
			g_rtSight.right++;
		}
	}
}


// 向下移动
void OnDown()
{
	if (g_ptPlayer.y < g_szMap.cy && GetMazeItem(g_ptPlayer.x, g_ptPlayer.y + 1) != MAP_WALL)
	{
		g_ptPlayer.y++;

		if (g_rtSight.bottom - g_ptPlayer.y < 8 && g_rtSight.bottom <= g_szMap.cy)
		{
			g_rtSight.top++;
			g_rtSight.bottom++;
		}
	}
}

//在地图中做标记
void OnMark(MAPITEM value)
{
	g_aryMap[g_ptPlayer.x][g_ptPlayer.y] = value;
}

// 检查是否到出口
bool CheckWin()
{
	if (g_ptPlayer.x == g_szMap.cx && g_ptPlayer.y == g_szMap.cy - 1)
	{
		// 创建对话框
		HWND hwnd = GetHWnd();
		if (MessageBoxW(hwnd, _T("恭喜你逃离了MAZE！\n您想再来一局吗？"), _T("恭喜"), MB_YESNO | MB_ICONQUESTION) == IDYES)
		{
			initGame();
			return false;
		}
		else
			return true;
	}

	return false;
}
