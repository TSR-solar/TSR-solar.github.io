介绍

学习C和C++的笔记！

从2024年10月29日开始，将会逐渐将代码和注释分离，并将注释用markdown文件重写；而代码会成为markdown笔记的附件；

本笔记前面大部分是C的学习内容，还有少量C++的拓展。后面C++的内容会多一些。已经在笔记中用标题做了区分，并用c和cpp后缀来区分代码附件。 所以，虽然C和C++交替出现，抛弃掉C++的拓展也能获得详尽的C语言笔记。

笔记没有完全按照学习顺序分布，而是模块化的；若感到理解困难，不妨跳到对应的模块查看相关内容，或自行上网查询资料；

由于编译时可能会出现中文乱码问题，因此代码中的输出均使用英文表示。

本笔记现已在GitHub和Gitee上发行。
